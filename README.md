# StudyFlow: AI-Powered Study Planner and Learning Analytics Dashboard

## Project Overview

StudyFlow is a Python web application built with Streamlit. The app helps students organize study tasks, manage deadlines, track progress, and analyze their workload.

The project includes a SQLite database, task management features, search/filter/sorting, a learning analytics dashboard, and an AI-powered study advisor.
